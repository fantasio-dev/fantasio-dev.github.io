# 토픽 페이지 표준 템플릿

## 개요

이 문서는 기술사 학습용 토픽 페이지의 **표준 양식**을 정의합니다.
모든 AI/DS/SEC/NW/DB/BIZ/CAOS/SW 토픽 페이지는 이 템플릿을 따라야 합니다.

## 필수 섹션

| 섹션 | 필수 여부 | 설명 |

# 지도학습(Supervised Learning)
{: .fs-8 }

1.2 지도학습
{: .label .label-purple }

---

## 핵심 키워드

`훈련데이터` `분류(Classification)` `회귀(Regression)`

---

## 정의/개념

훈련데이터로부터 함수를 유추하는 기계학습 방법

---

## 개념도

```
기계학습 (Machine Learning)
├── 지도학습 (Supervised Learning) ⭐
│   ├── 분류: SVM, KNN, CNN, RNN
│   └── 회귀: 선형회귀, 신경망
├── 비지도학습 (Unsupervised Learning)
└── 강화학습 (Reinforcement Learning)
```

---

## 구성요소 `분회` / `스케씨알` / `선신`

### 분류(Classification) 알고리즘 `스케씨알`
{: .highlight-purple }

| 암기 | 항목 | 설명 |
|:-----|:-----|:-----|
| **스** | SVM | 경계선 근접데이터(SV) 분석 |
| **케** | KNN | 거리기반 클러스터 할당 |
| **씨** | CNN | Convolution, Pooling layer |
| **알** | RNN | 셀에 정보저장, 은닉층 |

### 회귀(Regression) 알고리즘 `선신`
{: .highlight-purple }

| 암기 | 항목 | 설명 |
|:-----|:-----|:-----|
| **선** | 선형회귀분석 | 종속/독립변수 관계 모델링 |
| **신** | 신경망 분석 | 다층 퍼셉트론 기반 비선형 회귀 |
