### 한 줄 정의

> **대형SW사업 전문평가제도**: 40억 이상 대형 SW사업에서 4개 전문영역별 전문가가 40%를 전담평가하는 제도

### 쉬운 비유

| 기존 평가 방식 | 개정된 전문평가 방식 |
|:------------|:-----------------|
| 모든 과목을 담임선생님 혼자 채점 | 수학은 수학선생님, 과학은 과학선생님이 채점 |
| 공통평가 100% | 공통평가 60% + 전문평가 40% |

### 왜 필요한가? (3줄 요약)

1. **평가 전문성 제고**: 복잡한 대형 SW사업에 전문가 심층 평가 필요
2. **공정경쟁 환경 조성**: 허위내용 판단절차 명확화, 타사 비방 시 불이익 부여
3. **중소기업 부담 경감**: 제안서 발표 기준금액 상향 (2.2억→5억)

### 3. 대형 소프트웨어 사업 전문평가 항목 및 기준

{: .highlight }
> **전문평가 4개 영역**: 정보기술개발, 정보보호, 데이터구축, 디지털기술
> - (정보기술개발) `전구운조` (전략, 구축, 운영, 조직역량)

### 4. 개정 시행 및 전문평가 실시에 따른 기대효과

{: .highlight }
> **기대효과**: 기술적, 경제적, 사회적, 사업적 4가지 측면
> - (기술) `기술력보유업체` - 기술용역도 포함해 기술력 보유 업체 선정
> - (경제) `비용절감상생` - 5억 이상만 발표, 중소업체 비용 절감
> - (사회) `공정경쟁` - 비방/허위 근절, 제도화 사회 달성
> - (사업) `품질향상` - 조달 전문성 제고, 사업 품질 향상

| 측면 | 기대효과 | 세부 내용 |
|:-----|:---------|:---------|
| **기술적 측면** | 기술력 보유 업체 선정 가능 | 용역분야를 기술용역도 포함시키도록 개정해 기술력 보유 업체 선정 가능 |
| | 사업 품질 고도화 | 기술력 보유 업체 참여에 따른 사업 품질향상 및 사업 성과 고도화 달성 |
| **경제적 측면** | 중소업체 비용 절감 | 5억원 이상 사업에 한해 발표를 의무화해 중소 업체 발표자료 작성비용 절감 |
| | 상생 생태계 촉진 | 대·중·소기업 상생 생태계 촉진 |
| **사회적 측면** | 공정경쟁 풍토 조성 | 업체간 비방, 허위사실 근절 통해 공정경쟁 풍토 조성 가능 |
| | 제도화 사회 달성 | 상위법규 개정사항 반영 및 조문 정비해 고도의 제도화 사회 달성 |
| **사업적 측면** | 제안평가 전문성 향상 | 조달청에 제안평가 요청 통해 제안평가 전문성 향상 가능 |
| | 사업 품질 향상 | 조달 전문성 제고해 진행 사업 품질 향상 가능 |
