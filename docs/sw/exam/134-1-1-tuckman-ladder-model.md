### 📖 등장배경

```
🌐 과거: 프로젝트 팀을 구성했는데 왜 처음부터 성과가 안 나올까?

💡 문제: "팀원들이 모이면 바로 일 잘하는 거 아니야? 왜 시간이 필요해?"

✅ 해결: 심리학자 터크만이 팀 발달에는 단계가 있다는 것을 발견!
   → 처음엔 서로 탐색하는 시기가 필요하고
   → 갈등을 겪으면서 서로를 이해하고
   → 규칙을 정하고 나서야 진짜 성과를 낼 수 있다!
```

---

### 📝 개념 정의

> **터크만 사다리 모델**: 팀이 형성부터 해산까지 거치는 5단계 발달 과정을 설명한 모델

| 구분 | 정의 (22자 이내) |
|:-----|:----------------|
| **Forming** | 팀이 처음 만들어지는 탐색 시기 |
| **Storming** | 의견 충돌이 발생하는 격동 시기 |
| **Norming** | 규칙을 정하고 협력하는 시기 |
| **Performing** | 팀이 최고 성과를 내는 시기 |
| **Adjourning** | 프로젝트 종료 후 해산 시기 |

### 쉬운 비유

| 팀 발달 단계 | 학교 반 비유 |
|:------------|:------------|
| Forming | 🏫 새 학기 첫날 - 서로 어색하게 인사하는 단계 |
| Storming | 😤 체육대회 준비 - "나는 이게 더 좋은데!" 의견 충돌 |
| Norming | 🤝 역할 분담 - "너는 응원, 나는 선수!" 협력 시작 |
| Performing | 🏆 체육대회 당일 - 완벽한 팀워크로 우승! |
| Adjourning | 👋 학기 끝 - "다음에 또 만나자!" 해산 |

---

### 🏗️ 기술요소 (2그룹, 각 그룹당 4가지)

#### 그룹 1: 단계별 주요 특성 `형격규성`

| 단계 | 팀원 상태 | 리더 역할 |
|:-----|:---------|:---------|
| **형**성기 | 서로 탐색, 불확실함 | 방향 제시 (지시형) |
| **격**동기 | 갈등 발생, 의견 충돌 | 중재자 역할 (지도형) |
| **규**범기 | 협력 시작, 규칙 수립 | 참여 유도 (참여형) |
| **성**과기 | 몰입, 높은 생산성 | 권한 위임 (위임형) |

#### 그룹 2: 단계별 핵심 이슈 `불의지관`

| 단계 | 딜레마 | 해결 과제 |
|:-----|:------|:---------|
| Forming | **불**확실성 | 팀 목표와 방향 명확화 |
| Storming | **의**견불일치 | 역할 명료화, 갈등 해결 |
| Norming | **지**원 vs 간섭 | 적절한 지원과 자율성 균형 |
| Performing | **관**심 vs 고립 | 고립된 팀원 없이 성과 극대화 |

---

### ⭐ 차별점 키워드

> **Storming 필수론**: 갈등(Storming) 단계를 건너뛰면 진정한 Performing에 도달할 수 없다!

## 🎯 기술사 수준 설명

{: .highlight }
> **터크만 사다리 모델**: 프로젝트 성과 달성 필요 → 팀 발달 단계, 단계별 리더십, 갈등 관리 → 5단계 팀 개발 모델
> - (단계) `FSNPA` (형격규성해)
> - (리더십) `지지참위지`
> - (딜레마) `불의지관상`
> - ⭐ **차별점**: Storming 필수론 - 갈등 단계를 극복해야만 진정한 Performing 도달 가능

---

### 🔑 핵심 키워드 3개

| 키워드 | 설명 |
|:------|:-----|
| **5단계 발달 모델** | Forming → Storming → Norming → Performing → Adjourning |
| **단계별 리더십** | 각 단계에 맞는 리더십 스타일 전환 필요 |
| **Storming 극복** | 갈등 단계를 건강하게 극복해야 팀 성과 향상 |

```

💡 문제: "팀의 성장 과정을 어떻게 체계적으로 이해하고 관리할 수 있을까?"
```

### 📝 개념 정의

| **핵심 개념** | 팀 발달 과정을 5단계로 표현 |

### 🏗️ 구성요소

#### 그룹 1: 5단계 발달 과정 `FSNPA`

| 단계 | 설명 | 주요 활동 |
|:-----|:-----|:---------|
| **F**orming (형성기) | 팀 구성원 상호 인식 및 탐색 | 팀 방향 설정, 역할 배정 |
| **S**torming (격동기) | 갈등 발생, 의견 불일치 | 계획 수립, 역할 명료화 |
| **N**orming (규범기) | 협력 관계 구축, 규범 형성 | 업무 및 역할 몰입 |
| **P**erforming (성과기) | 생산성 향상, 목표 달성 | 수행관리 및 평가 |
| **A**djourning (해산기) | 과업 종료, 팀 해산 | Lessons Learned 정리 |

#### 그룹 2: 단계별 특성 매트릭스 `지지참위지`

| 단계 | 과업 목표 | 관계상 목표 | 필요 리더십 |
|:-----|:---------|:-----------|:-----------|
| **F**orming | 열성 | 수용 | **지**시형 |
| **S**torming | 역할 명료화 | 소속감 | **지**도형 |
| **N**orming | 몰입 | 지원 | **참**여형 |
| **P**erforming | 성취 | 자긍심 | **위**임형 |
| **A**djourning | 인도물 종료 | 팀관계 정리 | **지**시형 |

### 🔄 단계별 주요 딜레마 및 대응

```
┌───────────────────────────────────────────────────────────────────┐
│  성과      ▲                                                      │
│            │                           ④ Performing              │
│            │                          ╱  (성과기)                 │
│            │                         ╱   - 관심 vs 고립           │
│            │   ① Forming            ╱                             │
│            │    (형성기)           ╱  ③ Norming    ⑤ Adjourning   │
│            │   - 불확실성        ╱    (규범기)      (해산기)       │
│            │        ╲           ╱    - 지원 vs 간섭 - 상실감       │
│            │         ╲         ╱                                   │
│            │          ╲       ╱                                    │
│            │           ╲     ╱                                     │
│            │      ② Storming                                      │
│            │       (격동기)                                        │
│            │      - 의견불일치                                     │
│            └──────────────────────────────────────────────▶ 시간   │
└───────────────────────────────────────────────────────────────────┘
```

| 단계 | 주요 딜레마 | 대응 전략 |
|:-----|:-----------|:---------|
| Forming | 불확실성 | 명확한 방향 제시, 팀 목표 설정 |
| Storming | 의견불일치 | 갈등 해결 기법 활용, R&R 명확화 |
| Norming | 지원 vs 간섭 | 적절한 자율성 부여, 규범 확립 |
| Performing | 관심 vs 고립 | 팀원 참여 독려, 성과 인정 |
| Adjourning | 상실감 | Lessons Learned, 성과 축하 |

---

### 🚀 Performing 단계 진입 촉진 기법

| 기법 | 설명 | 효과 |
|:-----|:-----|:-----|
| **Ground Rule** | 규범(Norm) 설정을 통한 팀 성과 저해 요소 제거 | Storming 단계 단축 |
| **RACI 차트** | Responsible, Accountable, Consulted, Informed 정의 | R&R 명확화로 갈등 감소 |
| **1 on 1 미팅** | 개인별 면담을 통한 이슈 조기 발견 | 갈등 예방 및 해결 |
| **OKR** | 목표(Objectives)와 핵심결과(Key Results) 정렬 | 팀 방향 통일 |

---

### ⭐ 차별점 키워드

| 항목 | 차별점 |
|:-----|:------|
| **Storming 필수론** | 갈등 단계는 회피가 아닌 극복의 대상 - 건강한 Storming이 성과의 열쇠 |
| **리더십 전환** | 단계별로 지시형 → 지도형 → 참여형 → 위임형 → 지시형으로 전환 필요 |
| **사다리(Ladder) 비유** | 한 단계씩 올라가야 정상(Performing)에 도달 - 단계 건너뛰기 불가 |
| **팀 개발 모델** | 개인 역량이 아닌 팀 전체의 발달 과정에 초점 |
