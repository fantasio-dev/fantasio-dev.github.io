## 🎯 기술사 수준 설명

### 📌 핵심 암기 (Quick Reference)

{: .highlight }
> **딥뷰**: 사람처럼 인지하는 시각 인공지능 기술
> - (특징) `영내실대` 영상인식, 내용파악, 실시간분석, 대용량분석
> - (기술요소-수집) `수시대` 시각정보, 대규모처리
> - (기술요소-예측) `예내예지` 내용분석, 예지형응용, 지원

## 🏗️ 기술 영역

### 특징 `영내실대` / 기술요소 `수시대` `예내예지`

#### 특징 `영내실대`
{: .highlight-purple }

| 암기 | 항목 | 설명 |
|:-----|:-----|:-----|
| **영** | 영상인식 | 사물과 사람 분리 |
| **내** | 내용파악 | 영상 속 내용 파악 |
| **실** | 실시간분석 | 실시간 영상 분석 |
| **대** | 대용량분석 | 대량 데이터 처리 |

#### 수집기반 기술요소 `수시대`
{: .highlight-purple }

| 암기 | 항목 | 설명 |
|:-----|:-----|:-----|
| **시** | 시각정보 | 시각데이터자산화, 빅데이터저장, API지원 |
| **대** | 대규모처리 | 처리파이프라인, 하이브리드스케쥴러(GPU) |

#### 예측기반 기술요소 `예내예지`
{: .highlight-purple }

| 암기 | 항목 | 설명 |
|:-----|:-----|:-----|
| **내** | 내용분석 | 이미지분석, 시각텍소노미, 시멘틱추론 |
| **예** | 예지형응용 | 예측기술, 실시간대응 |
| **지** | 지원 | 플랫폼 지원 |
