## 🎯 기술사 수준 설명

### 📌 핵심 암기 (Quick Reference)

{: .highlight }
> **추천시스템**: 특정 시점에 특정 사용자가 관심을 가질만한 리스트를 찾아주는 시스템
> - (알고리즘) `컨협딥` 컨텐츠기반, 협업필터링, 딥러닝융합

## 🏗️ 기술 영역

### 추천 알고리즘 `컨협딥` / 평가지표

#### 추천 알고리즘 `컨협딥`
{: .highlight-purple }

| 암기 | 항목 | 설명 |
|:-----|:-----|:-----|
| **컨** | 컨텐츠기반 | 사용 기록 기반, 첫 사용자 불가 (Cold Start) |
| **협** | 협업필터링 | 타 사용자 사용 기록 기반 |
| **딥** | 딥러닝융합 | 딥러닝 기반 연구 중 |

#### 평가지표
{: .highlight-purple }

| 구분 | 지표 | 설명 |
|:-----|:-----|:-----|
| **비즈니스** | 매출, CTR | Click Through Rate |
| **품질지표** | 연관성 | 추천 아이템의 관련성 |
| **품질지표** | 다양성 | 추천 리스트의 다양성 |
| **품질지표** | 새로움 | 새로운 아이템 추천 |
| **품질지표** | 불예측성 | 예상치 못한 추천 |
| **품질지표** | 커버리지 | 전체 아이템 중 추천 비율 |

</div>

## ⭐ 차별점 키워드 (가산점 포인트)

{: .important }
> **추천시스템 핵심**
> - **Cold Start**: 컨텐츠 기반의 첫 사용자 문제
> - **협업 필터링**: User-based, Item-based
> - **하이브리드**: 두 방식 결합으로 단점 보완
