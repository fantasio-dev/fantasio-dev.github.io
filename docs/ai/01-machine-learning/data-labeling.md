## 🎯 기술사 수준 설명

### 📌 핵심 암기 (Quick Reference)

{: .highlight }
> **데이터 레이블링**: AI 학습에 사용될 수 있는 데이터를 필요 형식에 맞게 분석하여 가공 표기
> - (유형) `이택객분` 이미지태깅, 텍스트정형화, 객체탐지, 분류
> - (고려사항) 정확한 형식에 맞는 분석, 가이드라인에 맞는 가공표기

---

## 핵심 키워드

`태깅` `어노테이션` `지도학습` `라벨`

---

## 정의/개념

AI 학습에 사용될 수 있는 데이터를 필요 형식에 맞게 분석하여 가공 표기

---

## 유형 `이택객분`

| 암기 | 유형 | 설명 | 예시 |
|:-----|:-----|:-----|:-----|
| **이** | 이미지 태깅 | 이미지에 태그 부착 | 사진에 '고양이' 태그 |
| **택** | 텍스트 정형화 | 텍스트 구조화 | 감정 분석, 개체명 인식 |
| **객** | 객체 탐지 | Bounding Box 표시 | 자율주행 객체 인식 |
| **분** | 분류 | 카테고리 분류 | 스팸/정상 메일 분류 |

---

## 레이블링 방법

| 방법 | 설명 |
|:-----|:-----|
| **수동 레이블링** | 사람이 직접 레이블 부여 |
| **자동 레이블링** | 알고리즘/모델 활용 |
| **반자동 레이블링** | 자동 + 사람 검수 |
| **크라우드소싱** | 다수 참여자 활용 |

---

## 고려사항

| 항목 | 설명 |
|:-----|:-----|
| **정확성** | 정확한 형식에 맞는 분석 |
| **일관성** | 가이드라인에 맞는 가공표기 |
| **품질 관리** | 검수 및 피드백 프로세스 |
