## 핵심 키워드

`과적합` `정규화` `앙상블` `무작위 비활성화`

---

## 정의/개념

각 iteration마다 무작위로 일부 unit을 무시하고 학습하여 과적합을 완화시키는 방법

---

## 동작 방식

```
[학습 시]
○ - ○ - ○ - ○    ○: 활성화된 뉴런
○ - ✕ - ○ - ✕    ✕: 비활성화된 뉴런 (랜덤)
○ - ○ - ✕ - ○

[테스트 시]
○ - ○ - ○ - ○    모든 뉴런 활성화
○ - ○ - ○ - ○    가중치 스케일링 적용
○ - ○ - ○ - ○
```

---

## 드랍아웃 vs 정규화

| 항목 | 드랍아웃 | 정규화 (L1/L2) |
|:-----|:---------|:---------------|
| **적용 시점** | 학습 중 | 학습 전/중 |
| **방식** | 뉴런 무작위 비활성화 | 가중치에 페널티 부여 |
| **효과** | 앙상블 효과 | 가중치 크기 제한 |

---

## 효과

| 효과 | 설명 |
|:-----|:-----|
| **과적합 방지** | 특정 뉴런에 의존 방지 |
| **앙상블 효과** | 여러 서브네트워크 학습 효과 |
| **일반화 향상** | 테스트 데이터에서 성능 향상 |
