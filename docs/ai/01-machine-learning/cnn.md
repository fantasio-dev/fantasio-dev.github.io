## 🎯 기술사 수준 설명

### 📌 핵심 암기 (Quick Reference)

{: .highlight }
> **CNN**: 합성곱층을 사용해 이미지의 공간정보를 유지하면서 feature를 추출하는 딥러닝 모델
> - (구성) `입합풀완출` 입력→합성곱→풀링→완전연결→출력
> - ⭐ **차별점**: 공간정보 유지, 파라미터 공유로 효율적 학습

---

## 핵심 키워드

`합성곱` `풀링` `특성맵` `필터` `공간정보`

---

## 정의/개념

합성곱층을 사용해 이미지의 공간정보를 유지하면서 feature를 추출하는 딥러닝 모델

---

## 구조 `입합풀완출`

| 암기 | 레이어 | 설명 |
|:-----|:-------|:-----|
| **입** | 입력층 | 이미지 입력 (채널 포함) |
| **합** | 합성곱층 | 필터로 특성 추출, Feature Map 생성 |
| **풀** | 풀링층 | 다운샘플링, 차원 축소 |
| **완** | 완전연결층 | 분류를 위한 층 |
| **출** | 출력층 | 분류 결과 출력 |

---

## 합성곱 연산

```
입력 이미지      필터(커널)       Feature Map
┌───┬───┬───┐   ┌───┬───┐       ┌───┬───┐
│ 1 │ 2 │ 3 │   │ 1 │ 0 │  →    │ 4 │ 5 │
├───┼───┼───┤   ├───┼───┤       ├───┼───┤
│ 4 │ 5 │ 6 │   │ 0 │ 1 │       │ 7 │ 8 │
├───┼───┼───┤   └───┴───┘       └───┴───┘
│ 7 │ 8 │ 9 │
└───┴───┴───┘
```

## 주요 개념

| 개념 | 설명 |
|:-----|:-----|
| **필터 (커널)** | 특성 추출용 가중치 행렬 |
| **스트라이드** | 필터 이동 간격 |
| **패딩** | 출력 크기 조절용 테두리 추가 |
| **채널** | 입력 깊이 (RGB = 3채널) |
