# 과적합(Overfitting)/과소적합(Underfitting)
{: .fs-8 }

기계학습

---

## 🎯 기술사 수준 설명

### 📌 핵심 암기 (Quick Reference)

{: .highlight }
> **과적합**: 학습데이터에만 높은 성능, 일반화 낮은 문제
> - (해결방안) `정조드데` 정규화, 조기종료, 드롭아웃, 데이터추가

---

## 핵심 키워드

`과적합` `과소적합` `정규화` `드롭아웃` `조기종료`

---

## 정의/개념

| 구분 | 정의 |
|:-----|:-----|
| **과적합 (Overfitting)** | 학습데이터에만 높은 성능, 일반화 낮은 문제 |
| **과소적합 (Underfitting)** | 학습이 부족하여 전반적으로 낮은 성능 |

---

## 과적합 vs 과소적합

| 항목 | 과적합 | 과소적합 |
|:-----|:-------|:---------|
| **학습 데이터 성능** | 높음 | 낮음 |
| **테스트 데이터 성능** | 낮음 | 낮음 |
| **원인** | 과도한 학습, 복잡한 모델 | 부족한 학습, 단순한 모델 |
| **모델 복잡도** | 과다 | 부족 |

---

## 과적합 해결방안 `정조드데`

| 암기 | 방안 | 설명 |
|:-----|:-----|:-----|
| **정** | 정규화 | L1/L2 정규화로 가중치 제한 |
| **조** | 조기종료 | Early Stopping, 검증 오차 증가시 종료 |
| **드** | 드롭아웃 | 일부 뉴런 랜덤 비활성화 |
| **데** | 데이터추가 | 학습 데이터 증강 (Data Augmentation) |

---

## 과소적합 해결방안

| 방안 | 설명 |
|:-----|:-----|
| **모델 복잡도 증가** | 더 많은 층/뉴런 추가 |
| **학습 시간 증가** | epoch 증가 |
| **특성 추가** | Feature Engineering |
