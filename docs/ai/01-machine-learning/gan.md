## 🎯 기술사 수준 설명

### 📌 핵심 암기 (Quick Reference)

{: .highlight }
> **GAN (Generative Adversarial Network)**: 적대적 생성 신경망, 생성자와 판별자 간의 경쟁을 통해 실제와 유사한 데이터 생성
> - (구성) 생성자, 판별자
> - ⭐ **차별점**: 두 네트워크 경쟁, 비지도학습 기반 이미지 생성

---

## 핵심 키워드

`생성자` `판별자` `적대적` `경쟁` `이미지 생성`

---

## 정의/개념

적대적 생성 신경망, 생성자와 판별자 간의 경쟁을 통해 실제와 유사한 데이터 생성

---

## 구성

| 구성 | 역할 | 목표 |
|:-----|:-----|:-----|
| **생성자 (Generator)** | 가짜 데이터 생성 | 판별자를 속이기 |
| **판별자 (Discriminator)** | 진짜/가짜 판별 | 정확히 구분하기 |

---

## 동작 원리

```
                    경쟁
        ┌──────────────────────┐
        │                      │
        ▼                      │
┌───────────┐            ┌───────────┐
│  Generator │ ──가짜──→ │Discriminator│
│  (생성자)  │            │  (판별자)   │
└───────────┘            └───────────┘
     ▲                         ▲
     │                         │
  랜덤 노이즈              실제 데이터
```

---

## 학습 과정

```
1. 생성자: 랜덤 노이즈로 가짜 이미지 생성
      ↓
2. 판별자: 실제 이미지와 가짜 이미지 구분 학습
      ↓
3. 생성자: 판별자를 속이도록 학습
      ↓
4. 반복하여 생성자 성능 향상
```

---

## GAN 활용

| 분야 | 활용 |
|:-----|:-----|
| **이미지 생성** | 얼굴, 예술 작품 생성 |
| **이미지 변환** | 스타일 변환 |
| **데이터 증강** | 학습 데이터 확장 |
| **초해상도** | 저해상도→고해상도 |
