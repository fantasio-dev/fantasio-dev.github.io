## 🎯 기술사 수준 설명

### 📌 핵심 암기 (Quick Reference)

{: .highlight }
> **PCA (Principal Component Analysis)**: 데이터 분산 기반 고차원 데이터를 저차원으로 축소하는 비지도학습
> - (프로세스) `정공주투` 정규화→공분산→주성분→투영
> - ⭐ **차별점**: 분산이 최대가 되는 축을 찾아 차원 축소

---

## 핵심 키워드

`차원축소` `분산` `고유벡터` `주성분` `공분산`

---

## 정의/개념

데이터 분산 기반 고차원 데이터를 저차원으로 축소하는 비지도학습

---

## 프로세스 `정공주투`

| 암기 | 단계 | 설명 |
|:-----|:-----|:-----|
| **정** | 정규화 | 데이터 표준화 (평균 0, 분산 1) |
| **공** | 공분산 행렬 | 변수 간 관계 계산 |
| **주** | 주성분 추출 | 고유값/고유벡터 계산 |
| **투** | 투영 | 주성분 축에 데이터 투영 |

---

## 주성분 개념도

```
        y
        │      ╱ PC1 (첫 번째 주성분)
        │    ╱      - 분산 최대 방향
    ●   ●  ╱
      ● ●╱ ●
       ╱● ●
     ╱  ●  ●
   ╱        
  ─────────── x
        │
        PC2 (두 번째 주성분)
        - PC1과 직교
```

---

## PCA 활용

| 분야 | 활용 |
|:-----|:-----|
| **차원 축소** | 고차원 데이터 저차원화 |
| **시각화** | 2D/3D 데이터 시각화 |
| **노이즈 제거** | 중요 특성만 보존 |
| **특성 추출** | 핵심 특성 추출 |
