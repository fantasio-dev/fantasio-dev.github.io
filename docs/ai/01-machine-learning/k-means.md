## 🎯 기술사 수준 설명

### 📌 핵심 암기 (Quick Reference)

{: .highlight }
> **K-Means**: 비지도학습 기반 군집 알고리즘, 데이터를 K개의 클러스터로 분류
> - (프로세스) `초할평반` 초기값 설정→할당→평균계산→반복
> - (한계) 초기값 의존, K값 설정 필요, 구형 클러스터만

## 핵심 키워드

`K` `중심점` `군집` `반복` `비지도학습`

## 정의/개념

비지도학습 기반 군집 알고리즘, 데이터를 K개의 클러스터로 분류

---

## 프로세스 `초할평반`

| 암기 | 단계 | 설명 |
|:-----|:-----|:-----|
| **초** | 초기값 설정 | K개의 초기 중심점 랜덤 선택 |
| **할** | 할당 | 각 데이터를 가장 가까운 중심점에 할당 |
| **평** | 평균계산 | 각 클러스터의 새 중심점 계산 |
| **반** | 반복 | 중심점 변화 없을 때까지 반복 |

---

## 알고리즘 시각화

```
[1단계: 초기 중심점]      [2단계: 할당]         [3단계: 새 중심점]
    ●                       ●○○                    ●
  ○ ○ ○                   ○                      ○ ○ ○
    ○                       ○                       
                           ■□□                    ■
  ○ ○                      □                      ○ ○
    ■                       □                       
                                                    
● ■: 중심점              ○ □: 각 클러스터
```

---

## K 값 결정 방법

| 방법 | 설명 |
|:-----|:-----|
| **Elbow Method** | SSE 그래프에서 꺾이는 점 |
| **Silhouette Score** | 클러스터 품질 지표 |
| **Gap Statistic** | 랜덤 분포와 비교 |
