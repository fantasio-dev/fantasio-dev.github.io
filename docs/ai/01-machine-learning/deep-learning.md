## 핵심 키워드

`DNN(심층신경망)` `CNN(다중신경망)` `RNN(순환신경망)` `여러개의 히든레이어`

---

## 정의/개념

다량의 레이블이 없는 원데이터로부터 데이터 부분들의 관계를 통해 레이블을 자동으로 생성하여 지도학습에 이용하는 학습 방법
- 컴퓨터가 마치 사람처럼 스스로 학습할 수 있는 인공지능 기술
- 고양이를 알려주지 않고 고양이를 분류해냄

---

## 등장배경

| 구분 | 내용 |
|:-----|:-----|
| **비지도학습기반 전처리기술발전** | 군집화 기반, 노이즈 감소 |
| **CNN 진화** | 가중치, 편향, Dropout |
| **RNN 진화** | 음성인식, 자연어 |
| **하드웨어 개선** | GPU 발전 |

## 알고리즘 `딘씬알`

| 암기 | 항목 | 설명 |
|:-----|:-----|:-----|
| **딘** | DNN(심층신경망) | 여러개의 히든레이어(다층 퍼셉트론) |
| **씬** | CNN(다중신경망) | 대비 + 차원축소 |
| **알** | RNN(순환신경망) | 신경망 연결에 순환구조, 히든레이어값 저장 후 이용 |
