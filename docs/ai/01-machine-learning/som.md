## 🎯 기술사 수준 설명

### 📌 핵심 암기 (Quick Reference)

{: .highlight }
> **SOM (Self-Organizing Map)**: 2D 그리드 맵 형태의 경쟁학습 알고리즘, 고차원 데이터를 저차원으로 시각화
> - (구성) 입력층, 경쟁층
> - (프로세스) 최근접 뉴런 탐색→이웃 뉴런 갱신→반복
> - ⭐ **차별점**: 비지도학습으로 데이터 시각화 및 클러스터링

---

## 핵심 키워드

`경쟁학습` `2D 그리드` `시각화` `차원축소`

---

## 정의/개념

2D 그리드 맵 형태의 경쟁학습 알고리즘, 고차원 데이터를 저차원으로 시각화

---

## 구조

| 구성 | 설명 |
|:-----|:-----|
| **입력층** | 고차원 입력 데이터 |
| **경쟁층** | 2D 그리드 뉴런 배열 |

---

## 학습 프로세스

```
1. 입력 데이터 제시
      ↓
2. 최근접 뉴런 (BMU) 탐색
      ↓
3. BMU와 이웃 뉴런 가중치 갱신
      ↓
4. 수렴할 때까지 반복
```

---

## SOM 개념도

```
입력 벡터 (고차원)
    │
    ▼
┌───────────────┐
│ ○ ○ ○ ○ ○ ○ │
│ ○ ○ ● ○ ○ ○ │  ← 2D 그리드 (경쟁층)
│ ○ ● ■ ● ○ ○ │     ■: BMU (Best Matching Unit)
│ ○ ○ ● ○ ○ ○ │     ●: 이웃 뉴런
│ ○ ○ ○ ○ ○ ○ │
└───────────────┘
```

---

## SOM 활용

| 분야 | 활용 |
|:-----|:-----|
| **데이터 시각화** | 고차원 데이터 2D 표현 |
| **클러스터링** | 유사 데이터 그룹화 |
| **이상 탐지** | 비정상 패턴 식별 |
