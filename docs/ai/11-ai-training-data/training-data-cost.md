## 비용산정 3요소

| 요소 | 설명 |
|:-----|:-----|
| **비용산정요소** | 비용발생항목, 물량, 인원, 기간, 장소, 장비 |
| **작업그룹요소** | 최소작업단위, 단가 |
| **데이터구축공정** | 구축유형(텍스트, 이미지, 음성, 비디오), 구축 프로세스 |

---

## 비용산정절차 `설공인제직최`

| 암기 | 단계 | 설명 |
|:-----|:-----|:-----|
| **설** | 작업설계 | 작업요소 정의, 물량 산정 |
| **공** | 투입공수산정 | 작업요소별 투입공수 |
| **인** | 직접인건비산정 | 투입공수 기초 인건비 산정 (수집, 생성, 가공) |
| **제** | 제경비산정 | 직접인건비 기초 제경비 산정 (115~120%) |
| **직** | 직접경비산정 | 콘텐츠 구매, 장비, 장소 |
| **최** | 최종비용산정 | 직접인건비 + 제경비 + 직접경비 |

---

## 비용 구성

```
최종비용 = 직접인건비 + 제경비 + 직접경비

├── 직접인건비: 수집, 생성, 가공
├── 제경비: 직접인건비 × 115~120%
└── 직접경비: 콘텐츠 구매, 장비, 장소
```
