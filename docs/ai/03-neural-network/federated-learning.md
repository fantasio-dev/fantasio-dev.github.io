#### 내용

- 분산된 환경에서 데이터의 직접적인 공유 없이 많은 로컬서버들이 중앙서버의 지시를 받아 훈련을 수행하는 학습
- > 정보보호: 분산서버가 데이터를 내보내지않음
- > 저지연필요없음:결과로 모델만 전송

#### 절차

- > 중앙서버>최적모델전송>Device 자체데이터로 학습>학습된 모델전송>중앙서버에서 Globalization

#### 분류

- > 수직: 데이터A 데이터B 영역, 수직으로 중첩
- > 수평: 데이터A 데이터B 영역, 수평으로 교차
- > 전이: 교차한 영역 라벨 전이

#### 학습환경

- > Cross Silo: 지역적으로 떨어진 센터, 1 > 1000
- > Cross Device: 수 많은 모바일/iot기기, 10^10이상
- * 통제는 하지만 데이터 접근 불가

## 🎯 기술사 수준 설명

### 📌 핵심 암기 (Quick Reference)

{: .highlight }
> **연합학습**: 분산된 환경에서 데이터 공유 없이 로컬에서 훈련하는 학습
> - (절차) `중모학전글` 중앙서버→모델전송→학습→전송→Globalization
> - (분류) 수직, 수평, 전이

---

<div class="exam-concept-block" markdown="1">

## 🧠 개념 영역

### 🔑 핵심 키워드 3개

| 키워드 | 설명 | 예시 |
|:------|:-----|:-----|
| **분산 학습** | 데이터를 중앙에 모으지 않고 분산 학습 | 모바일 기기 |
| **정보 보호** | 분산 서버가 데이터를 내보내지 않음 | 개인정보 보호 |
| **모델 집계** | 학습된 모델만 전송 | 결과만 공유 |

---

### 📖 등장배경

| **분산 환경** | 수많은 기기에 분산된 데이터 활용 필요 |

## 🏗️ 기술 영역

### 절차 `중모학전글` / 분류 / 학습환경

#### 절차 `중모학전글`
{: .highlight-purple }

| 암기 | 항목 | 설명 |
|:-----|:-----|:-----|
| **중** | 중앙서버 | 초기 모델 관리 |
| **모** | 최적모델전송 | Device로 모델 전송 |
| **학** | 자체데이터로 학습 | 로컬에서 학습 수행 |
| **전** | 학습된 모델전송 | 중앙서버로 모델 전송 |
| **글** | Globalization | 중앙서버에서 모델 통합 |

#### 분류
{: .highlight-purple }

| 유형 | 설명 |
|:-----|:-----|
| **수직** | 데이터A 데이터B 영역, 수직으로 중첩 |
| **수평** | 데이터A 데이터B 영역, 수평으로 교차 |
| **전이** | 교차한 영역 라벨 전이 |

#### 학습환경
{: .highlight-purple }

| 유형 | 설명 |
|:-----|:-----|
| **Cross Silo** | 지역적으로 떨어진 센터, 1~1000 |
| **Cross Device** | 수많은 모바일/IoT 기기, 10^10 이상 |

</div>
