### 🔑 핵심 키워드 3개

| 키워드 | 설명 |
|:------|:-----|
| **응집도(Cohesion)** | 같은 군집 안에서 얼마나 "가깝게" 뭉쳤는가 |
| **분리도(Separation)** | 다른 군집과 얼마나 "멀리" 떨어졌는가 |
| **S(i)** | 한 점이 "자기 군집에 잘 속했는지" 점수(-1~1) |

### 📖 등장배경

| 구분 | 내용 |
|:-----|:-----|
| **과거** | 클러스터링은 정답 레이블이 없어서 "잘 됐는지" 평가가 어려움 |
| **문제** | 군집 개수 K 선택이나 알고리즘 비교를 정량 지표로 하고 싶음 |

### 📝 개념 정의

| 개념 | 정의 | 쉬운 비유 |
|:-----|:-----|:---------|
| **실루엣 계수** | 응집도/분리도를 함께 평가하는 지표 | 내 친구들과 가까운지, 다른 모임과는 먼지 |

### 🧩 기술요소 (2그룹 × 4개)

#### 그룹 1: 계산 요소 `군최점평`

| 요소 | 설명 |
|:-----|:-----|
| **군**집 내 평균거리 a(i) | 같은 군집 내 다른 점들과의 평균 거리 |
| **최**근접 군집 평균거리 b(i) | 가장 가까운 다른 군집과의 평균 거리 |
| **점**수 S(i) | (b-a)/max(a,b) 공식으로 계산 |
| **평**균 S̄ | 전체 데이터의 S(i) 평균 |

#### 그룹 2: 해석/활용 `범클최알`

| 요소 | 설명 |
|:-----|:-----|
| **범**위 | -1 ~ 1 (1에 가까울수록 좋음) |
| **클**러스터 품질 | Clear-cut/Weak cluster 판단 |
| **최**적 K 선택 | 여러 K 비교 후 S̄ 높은 K 선택 |
| **알**고리즘 비교 | K-means vs GMM 등 정량 비교 |

### ⭐ 차별점 키워드

**"라벨 없이 품질 평가"** - 비지도 학습에서도 군집 품질을 빠르게 정량 평가 가능

### 🧾 6하원칙 요약

| 항목 | 내용 |
|:-----|:-----|
| **누가** | 각 데이터 포인트 i가 |
| **언제** | 클러스터링 결과가 나온 뒤 |
| **어디서** | 자기 군집과 가장 가까운 다른 군집을 기준으로 |
| **무엇을** | 응집도/분리도를 |
| **왜** | 군집 품질과 최적 K를 판단하기 위해 |
| **어떻게** | a(i), b(i) → (b-a)/max(a,b)로 계산 |

### 📌 핵심 암기 (Quick Reference)

{: .highlight }
> **실루엣 계수**: 군집 품질 객관적 평가 필요 → 응집도, 분리도, -1~1 → 군집 품질 평가 지표
> - (정의) `응분` - 응집도+분리도 통합 평가
> - (공식) `바아맥` - S(i)=(b-a)/max(a,b)
> - (활용) `최K비이` - 최적K, 비교, 이상치 탐지
> - ⭐ **차별점**: 라벨 없이 품질 평가 (비지도 학습 평가): > - 정답 레이블 없이도 군집 품질 정량 평가 가능

#### 그룹 1: 계산 요소 `군최점평`
{: .highlight-purple }

| 암기 | 요소 | 수식/설명 |

| **군** | 군집 내 평균거리 a(i) | 포인트 i와 동일 군집 내 다른 포인트들의 평균 거리 |
| **최** | 최근접 군집 평균거리 b(i) | 포인트 i와 가장 가까운 다른 군집 포인트들의 평균 거리 |

## ⭐ 차별점 키워드 (가산점 포인트)

{: .important }
> **라벨 없이 품질 평가 (비지도 학습 평가)**
> - 정답 레이블 없이도 군집 품질 정량 평가 가능
> - 시간복잡도 O(n²) 주의 (대규모 데이터 시 샘플링 필요)
> - 밀도 기반 클러스터링(DBSCAN)에는 부적합할 수 있음

<details markdown="1">
<summary><h3 style="display:inline">🧠 상세 설명 (클릭해서 펼치기)</h3></summary>

#### 📖 등장배경 상세

```
🌐 과거: 클러스터링은 정답 레이블이 없어서 "잘 됐는지" 평가가 어려움

💡 문제: 군집 개수 K 선택이나 알고리즘 비교를 정량 지표로 하고 싶음

✅ 해결: 군집 내 거리(응집)와 최근접 군집 거리(분리)를 함께 보는 실루엣 계수
   → 각 포인트별 S(i) 계산 후 평균
   → -1~1 사이 값으로 품질 판단
```

---

#### 📋 수식 (개별 포인트 i)

$$S(i)=\frac{b(i)-a(i)}{\max(a(i),b(i))}$$

| 기호 | 의미 |
|:-----|:-----|
| **a(i)** | 포인트 i와 **동일 군집 내 다른 포인트**들과의 평균 거리 (응집도) |
| **b(i)** | 포인트 i와 **가장 가까운 다른 군집** 포인트들과의 평균 거리 (분리도) |

---

#### 📋 계산 프로세스

| 프로세스 | 핵심요소 | 설명 |
|:---------|:---------|:-----|
| 개별 포인트간 거리 측정 | 유클리디안 거리 | 같은 군집/다른 군집 거리 계산 |
| 군집 내 평균거리 a(i) | 내부 군집 응집도 | i와 동일 군집 내 포인트 평균 거리 |
| 최인접 군집 평균거리 b(i) | 외부 군집 분리도 | i와 가장 가까운 다른 군집 평균 거리 |
| 실루엣 계수 S(i) 계산 | -1~1 | 수식으로 S(i) 산출 |
| 평균 실루엣 계수 S̄ | 군집 품질 평가 | 전체 포인트 S(i) 평균 |
| 최적 K 도출 | 최적 K | 여러 K 비교 후 S̄가 큰 K 선택 |

---

#### 📋 장단점

| 구분 | 설명 |
|:-----|:-----|
| **장점** | 알고리즘 영향 적음(결과로 평가), 적정 K 선택 기준 제공, 결과 시각화/해석 용이 |
| **단점** | 모든 포인트 거리 계산 필요로 O(n²), 평균만으로 판단 어려움, 밀도 기반에 부적합 |

---

#### 📈 상위 토픽 계층도

```
군집 평가 (Cluster Evaluation)
├── 내부 평가 (Internal Evaluation)
│   ├── 실루엣 계수 (Silhouette Coefficient) ⭐
│   ├── Dunn Index
│   ├── Davies-Bouldin Index
│   └── Elbow Method
├── 외부 평가 (External Evaluation)
│   ├── Adjusted Rand Index (ARI)
│   ├── Normalized Mutual Information (NMI)
│   └── Fowlkes-Mallows Index
└── 관련 알고리즘
    ├── K-Means
    ├── GMM (Gaussian Mixture Model)
    └── DBSCAN
```

### 📊 S(i) 값 해석

| 구분 | 값 | 해석 |
|:-----|:--:|:-----|
| 개별 S(i) | ≈ 1 | 최적 (해당 군집에 잘 속함) |
|  | ≈ 0 | 경계 (군집 간 거리가 비슷) |
|  | ≈ -1 | 잘못됨 (이웃 군집이 더 가까움) |
| 평균 S̄ | 높음 | Clear-cut cluster (군집 명확) |
|  | 낮음 | Weak cluster (군집 약함) |
