### 🔑 핵심 키워드 3개

| 키워드 | 설명 |
|:------|:-----|
| **모달리티(Modality)** | 텍스트·이미지·음성·비디오 같은 "입력 종류" |
| **융합(Fusion)** | 서로 다른 모달 정보를 합쳐서 이해/추론 |
| **정렬(Alignment)** | 모달 간 의미를 맞춰 공통 표현 공간을 만드는 것 |

### 📖 등장배경

| 구분 | 내용 |
|:-----|:-----|
| **과거** | 텍스트만(언어모델), 이미지만(CNN)처럼 단일 모달(Unimodal) 위주로 발전 |
| **문제** | 실제 세계는 "보고 + 듣고 + 읽고"가 동시에 일어나는데, 단일 모달은 상황 이해가 제한됨 |

### 📝 개념 정의

| 개념 | 정의 | 쉬운 비유 |
|:-----|:-----|:---------|
| **멀티모달 AI** | 텍스트·이미지·음성·비디오 등 다중 모달리티를 통합 처리하는 인공지능 | 사람처럼 보고, 듣고, 읽고 종합 이해하는 AI |

### 🧩 기술요소 (2그룹 × 4개)

#### 그룹 1: 처리 기술 `지음이추`

| 요소 | 설명 |
|:-----|:-----|
| **지**식/언어지능 | NLP/NLU/NLG 등 자연어 처리 |
| **음**성/청각 | STT, 신호처리 등 음성 처리 |
| **이**미지/시각 | 전처리, 객체 인식 등 비전 처리 |
| **추**론/기계학습 | 회귀분석, 클러스터링 등 |

#### 그룹 2: 핵심 요소 `융정표추`

| 요소 | 설명 |
|:-----|:-----|
| **융**합(Fusion) | Early/Late/Cross-Modal 결합 |
| **정**렬(Alignment) | 모달 간 의미 일치화 |
| **표**현 학습 | Embedding/Transformer 기반 |
| **추**론 강화 | MM-RAG/CoT 등 |

### ⭐ 차별점 키워드

**"사람형 인터페이스"** - 텍스트+이미지+음성의 결합으로 더 자연스러운 의사소통/상황 이해 가능

### 🧾 6하원칙 요약

| 항목 | 내용 |
|:-----|:-----|
| **누가(Who)** | 멀티모달 AI 모델이 |
| **언제(When)** | 입력이 들어올 때마다 |
| **어디서(Where)** | 여러 모달(텍스트/이미지/음성/비디오)에서 |
| **무엇을(What)** | 정보를 뽑아(표현) |
| **왜(Why)** | 더 정확한 상황 이해와 추론을 위해 |
| **어떻게(How)** | `지음이추`로 처리하고 `융정표추`로 융합/정렬해 결과 생성 |

> **멀티모달 기술요소**: 모달처리, 융합·정렬, 추론강화 → 다중 입력 통합 AI 핵심 기술
> - (처리기술) `지음이추` - 지식/언어, 음성/청각, 이미지/시각, 추론/기계학습
> - (융합기술) `얼레크` - Early, Late, Cross-Modal Attention
> - (리스크) `입융추출` - 입력→융합→추론→출력 단계별 취약점

## 🏗️ 기술 영역

<div class="exam-tech-block" markdown="1">

### 🏗️ 구성요소 (핵심 암기법 상세) `지음이추` / `융정표추`

#### 그룹 1: 처리 기술(모달별 요소기술) `지음이추`

| 암기 | 처리기술 | 요소기술 | 설명 |
|:-----|:---------|:---------|:-----|
| **지** | 지식/언어지능 | NLP, NLU, NLG, Word Embedding, Seq2Seq | 단어 분리/유형 라벨링, 검색, 의미/관계 추출, 자연어 이해/생성 |
| **음** | 음성/청각 | STT(Speech-to-Text), Signal Processing | 음성 인식, 핫워드, 노이즈 필터링 등 신호 전처리 |
| **이** | 이미지/시각 | Image Scaling, Filtering/Morphology | 세분화/이해, 객체·얼굴 인식, 이미지 기반 검색·예측 |
| **추** | 추론/기계학습 | 회귀분석/시계열분석, 클러스터링/연관분석 | KPI 예측/분석, 자동 데이터 생성, 추론 강화 기반 |

#### 그룹 2: 멀티모달 핵심 요소(융합·정렬·추론) `융정표추`

| 암기 | 핵심 기술요소 | 상세 설명 |
|:-----|:-------------|:---------|
| **융** | 융합(Fusion) | Early Fusion(입력 단계 결합), Late Fusion(결과 결합), Cross-Modal Attention(모달 간 상관 학습) |
| **정** | 의미 정렬(Alignment) | 모달 간 의미 불일치(misalignment) 해결, 공통 표현 공간 형성(semantic alignment) |
| **표** | 표현 학습(Representation) | 멀티모달 임베딩/Transformer 기반 표현(예: CLIP 계열 등) |
| **추** | 추론 강화(Inference) | MCoT(멀티모달 CoT), MM-RAG(멀티모달 근거 검색·결합)로 신뢰성/추론력 강화 |

## ⭐ 차별점 키워드 (가산점 포인트)

{: .important }
> **사람형 인터페이스와 Cross-Modal 이해**
> - 텍스트+이미지+음성의 결합으로 더 자연스러운 의사소통/상황 이해 가능
> - 인간의 오감 기반 상호작용을 모델이 흉내내는 방향으로 발전
> - GPT-4V, Gemini, Claude 3 등 최신 LLM이 멀티모달 지원

✅ 해결: 여러 모달을 동시에 처리해 상황 인식·추론·생성까지 수행하는 멀티모달 AI가 확산
   → CLIP, ImageBind 등 공통 표현 공간 활용

#### 📊 멀티모달 모델 사례(입력→출력)

| 입력 모달 | 출력 모달 | 모델/사례 |
|:---------|:---------|:---------|
| Text, Image | Image | 이미지 생성/편집(Stable Diffusion 등) |
| Image | Text | Image Captioning/Description(BLIP 등) |
| Image, Text | Text | Visual QA(Flamingo, GPT-4 계열) |
| Text | Video | Video Generation(Dreamix 등) |
| Text | Audio | Music Generation(MusicLM 등) |
| Speech, Text | Speech/Text | 음성 번역·STT·TTS(SeamlessM4T 등) |
| Robot sensors, Image, Text | Actions | 로봇 조작(PaLM-E 등) |

### 📊 융합 기술 `얼레크`

| 암기 | 융합 방식 | 설명 | 특징 |
|:-----|:---------|:-----|:-----|
| **얼** | Early Fusion | 입력 단계에서 모달 결합 | Concat input, 초기 정보 통합 |
| **레** | Late Fusion | 출력 단계에서 결과 결합 | Decision fusion, 독립 처리 후 결합 |
| **크** | Cross-Modal Attention | 모달 간 상관관계 학습 | Attention 기반, 상호작용 학습 |

### 📊 보안 취약점 및 대응(단계별) `입융추출`

| 암기 | 단계 | 보안 취약점 | 대응 방안 |
|:-----|:-----|:-----------|:---------|
| **입** | 입력 단계 | 적대적 예제 공격, 데이터 포이즈닝 | 적대적 훈련, 입력 데이터 무결성 검증/이상탐지 |
| **융** | 융합 단계 | 크로스모달 공격, 특징 정렬 오류 | Cross-modal 정규화, alignment 검증 손실 함수 추가 |
| **추** | 추론 단계 | 스테가노그래피 프롬프트 주입, 모델 탈취 | 프롬프트 검증/필터, 호출 제한(과도 쿼리 탐지/차단) |
| **출** | 출력 단계 | 정보유출(Privacy Leakage), 환각(Hallucination) | 차등 프라이버시/민감정보 최소화, 신뢰도 보정 |
