### 한 줄 정의
> **양자 머신러닝(QML)**: 양자컴퓨팅의 원리(중첩·얽힘·간섭)를 머신러닝에 접목하여 **기존 고전적 머신러닝의 한계를 극복**하고자 하는 첨단 융합 기술

### 쉬운 비유
- 기존 ML = "한 번에 하나씩 문을 열어 답을 찾는 것"
- QML = "모든 문을 동시에 열어 답을 찾는 것" (중첩·병렬성 활용)

### 왜 중요한가? (3줄 요약)
- AI 연산량이 기하급수적으로 증가하면서 **고전 컴퓨팅의 한계**에 직면
- 양자역학의 **중첩·얽힘·간섭**을 활용하면 특정 연산에서 **지수적 속도 향상** 가능
- 현재 NISQ(Noisy Intermediate-Scale Quantum) 단계이지만, **미래 AI 연산 혁신**의 핵심 기술로 부상

### 1. 양자 머신러닝(QML)의 개요

{: .highlight }
> **QML 개요**: 양자컴퓨팅, 머신러닝, 융합기술
> - (정의) `양자+ML`
> - (원리) `중얽간`
> - (효과) `병렬+지수`

#### 가. 양자 머신러닝(QML)의 정의

| 구분 | 설명 |
|:-----|:-----|
| **정의** | 머신러닝의 데이터 처리·학습 과정에 **양자 알고리즘 또는 양자 하드웨어를 접목**한 AI 학습기법 |
| **특징** | 양자역학의 핵심 원리인 **중첩, 얽힘, 간섭**을 활용하여 데이터 처리와 학습에서 새로운 가능성을 제시 |

#### 나. 양자컴퓨팅의 핵심 원리 `중얽간`

| 원리명 | 주요 내용 | 양자컴퓨팅에서의 역할 및 효과 |
|:------|:---------|:-------------------------|
| **중**첩 (Superposition) | 큐비트가 0과 1의 상태를 동시에 가질 수 있는 양자역학적 현상 | 병렬 연산 가능, 여러 계산을 동시에 수행하여 연산 속도 극대화 |
| **얽**힘 (Entanglement) | 두 개 이상의 큐비트가 서로 강하게 연결되어 한 큐비트의 상태가 다른 큐비트에 즉각 영향 | 큐비트 간 상관관계로 복잡한 문제를 효율적으로 처리, 정보 전달력 강화 |
| **간**섭 (Interference) | 여러 양자 상태가 서로 영향을 주어 특정 결과의 확률을 증폭하거나 감소시키는 현상 | 원하는 계산 결과의 확률을 높이고, 불필요한 결과는 억제 |

---

### 2. 양자 머신러닝의 주요 기술 및 알고리즘

{: .highlight }
> **주요 기술**: 데이터 전처리, 모델 학습, 측정/평가
> - (전처리) `인코딩`
> - (학습) `변커하`
> - (측정) `샘측포손`

#### 가. 양자 머신러닝의 주요 기술 (학습 단계별)

##### 그룹 1: 모델 학습 기술 `인변커하`

| 학습단계 | 주요기술 | 설명 |
|:--------|:--------|:-----|
| **①** 데이터 전처리 | 양자 데이터 **인**코딩 (Quantum Feature Embedding) | 고전 데이터를 양자 상태로 변환하여 큐비트에 학습데이터 반영 |
| **②** 모델 학습 | **변**분 양자 회로 (Variational Quantum Circuits, VQC) | 양자 회로 + 고전 최적화 알고리즘 결합 |
| **②** 모델 학습 | 양자 **커**널 방법 (Quantum Kernel Methods) | SVM 등 커널 기반 알고리즘과 결합, 양자 회로를 사용해 데이터를 양자 상태(Hilbert 공간)에 임베딩 |
| **②** 모델 학습 | **하**이브리드 양자-고전 구조 | 양자 회로는 특징 추출 담당, 고전 컴퓨터는 최적화 담당 |

##### 그룹 2: 측정/평가 기술 `샘측포손`

| 단계 | 주요기술 | 설명 |
|:----|:--------|:-----|
| **③** 측정, 평가 | 양자 **샘**플링 (Quantum Sampling) | 양자 상태의 확률 분포를 추정하고 예측값을 도출 |
| **③** 측정, 평가 | Quantum **Measurement** (**측**정) | 양자 상태 예측값을 고전적 데이터로 변환 |
| **③** 측정, 평가 | **Po**st-processing (**포**스트프로세싱) | 고전 알고리즘으로 결과를 해석/분류 |
| **③** 측정, 평가 | **손**실함수 | 양자 회로 출력의 예측값과 실제값을 비교해 계산 |

#### 나. 양자 머신러닝의 알고리즘

##### 그룹 1: 신경망 최적화 알고리즘 `HQV`

| 구분 | 알고리즘 | 설명 |
|:----|:--------|:-----|
| 신경망 최적화 | **H**HL 알고리즘 (Harrow-Hassidim-Lloyd) | 회귀분석, 신경망 학습, 최적화에 활용 가능 |
| 신경망 최적화 | **Q**AOA (Quantum Approximate Optimization Algorithm) | 머신러닝 모델의 파라미터 최적화에 응용, 조합 최적화 문제(스케줄링, 경로 탐색, 포트폴리오 선택 등)에 활용 |
| 신경망 최적화 | **V**QE (Variational Quantum Eigensolver) | 손실함수 최적화, 파라미터화된 양자 회로(ansatz)를 실행 → 결과를 고전 컴퓨터 최적화기로 피드백 → 반복 학습 |

##### 그룹 2: 연산효율화 및 탐색 알고리즘 `SKP볼강`

| 구분 | 알고리즘 | 설명 |
|:----|:--------|:-----|
| 연산 효율화 및 가속화 | Quantum **S**VM (Support Vector Machine) | 양자 커널을 이용해 고차원 특징 공간에서 분류 문제 해결 |
| 연산 효율화 및 가속화 | Quantum **k**-NN | 양자 상태 간 거리 계산을 통해 최근점 이웃 분류 가속화 |
| 연산 효율화 및 가속화 | Quantum **P**CA | 고차원 데이터 차원 축소에 활용 |
| 탐색 및 학습 효율 개선 | Quantum **볼**츠만 Machine (QBM) | 확률적 생성 모델을 양자로 구현 |
| 탐색 및 학습 효율 개선 | 양자 **강**화학습 (Quantum Reinforcement Learning) | 상태·행동 공간을 양자 상태로 확장 |

### 3. 양자 머신러닝과 기존 머신러닝의 비교

{: .highlight }
> **비교**: 계산단위, 임베딩, 표현방식, 복잡도, 한계점, 적용현황, 연산방식
> - (비교) `계임표복한적연`

#### 가. 개념 비교

| 구분 | 양자 머신러닝 | 기존 머신러닝 |
|:----|:-------------|:-------------|
| **개념** | 양자컴퓨터의 계산 원리를 **이용**해 머신러닝 알고리즘을 구현·가속화하는 AI 학습기법 | 알고리즘을 **이용**해 데이터를 분석하고 분석을 통해 스스로 학습을 하며, 학습한 내용을 기반으로 판단, 예측하는 학습 |

#### 나. 상세 비교 `계임표복한적연`

| 비교 | 양자 머신러닝 | 기존 머신러닝 |
|:----|:-------------|:-------------|
| **계**산단위 | 큐비트(qubit) | 비트(bit) |
| **임**베딩 | 큐비트 상태 변환 과정 필요 | 데이터 바로 사용가능 |
| 데이터 **표**현방식 | 양자상태로 표현 | 벡터/행렬 형태로 표현 |
| 계산**복**잡도 | 다항식 시간으로 감소가능 | 데이터에 따라 지수적 증가 |
| **한**계점 | 하드웨어 한계(NISQ 단계), 노이즈·오류 보정 필요 | 차원의 저주, 지수적 연산량 |
| **적**용현황 | 연구 PoC 단계 | 산업전반 사용중 |
| **연**산방식 | 병렬적 연산 | 순차적 연산 |

### 4. 양자 머신러닝 활성화 전략

{: .highlight }
> **활성화**: 기술 고도화, 산업 생태계, 국가 지원
> - (전략) `기생국`

| 전략 | 세부 활성화전략 | 설명 |
|:----|:-------------|:-----|
| **양자 머신러닝 기술 고도화** | 노이즈 억제 및 양자 에러 보정(QEC) 기술 개발 | 양자 상태가 쉽게 깨져서 학습 안정성 저하되므로 안정성 강화 및 오류정정 기술 확보 |
| **양자 머신러닝 기술 고도화** | 초전도체 플랫폼 이용, 외형적인 큐비트수 증가 | 큐비트 칩 간의 연결을 통해 대규모 큐비트 수를 확보하여 양자적 우위 입증 |
| **관련 산업 생태계 활성화** | QML 알고리즘 벤치마크와 성능 비교 지표 표준화 | 재현성 검증 강화 및 연구간 비교통해 산학연 협력의 기반 조성 |
| **관련 산업 생태계 활성화** | 양자 머신러닝 산업 조성 | 스타트업 육성과 투자 활성화를 중심으로 한 정책적 지원과 산업 기반 조성 |
| **국가차원의 연구개발 지원** | 양자컴퓨팅과 AI 융합 R&D 지원 확대하여 기술 경쟁력 확보 | 공공-민간 협력 기반의 공동 연구개발 플랫폼을 구축하고, 실증 및 상용화를 위한 테스트베드 조성 |
| **국가차원의 연구개발 지원** | 융합형 인재양성 및 확보 정책 마련 | 여러 분야의 지식과 다양한 전문성을 바탕으로 기술 개발에 나설 수 있도록 다학제간 교과목 및 실무형 재교육 프로그램 마련 |

## ⭐ 차별점 키워드 (가산점 포인트)

{: .important }
> **양자 우위(Quantum Advantage) 관점**
> - 현재 양자 머신러닝은 **NISQ(Noisy Intermediate-Scale Quantum)** 단계
> - 오류율이 높고 안정성이 부족하여 대부분 **PoC 수준**
> - 상용화를 위해 **기술·생태계·국가차원의 선결과제**가 남아있음
