```
💡 문제: "표본으로 구한 평균이 전체 평균과 얼마나 비슷할까?"

✅ 해결: 불편추정량을 사용하면 평균적으로 정확하게 추정!
   → 표본평균은 모평균의 불편추정량
   → 표본분산은 n-1로 나눠야 모분산의 불편추정량
   → 편향 없이 정확한 추정 가능!
```

### 📝 개념 정의

> **불편추정량(Unbiased Estimator)**: 추정량의 기대값(평균값)이 실제 모집단의 모수와 동일한 추정량

| 개념 | 쉬운 비유 |
|:-----|:---------|
| **불편추정량** | 과녁의 중앙을 평균적으로 정확히 맞추는 것 (흩어질 수 있지만 중심은 정확) |
| **편향추정량** | 과녁 중심에서 벗어난 곳을 평균적으로 맞추는 것 |

---

### 🏗️ 기술요소 (2그룹 × 4개)

#### 그룹 1: 점추정 4가지 준거 `불효일충`

| 요소 | 설명 |
|:-----|:-----|
| **불**편성 | 기대값이 모수와 일치 |
| **효**율성 | 분산이 가장 작음 |
| **일**치성 | n이 커지면 모수에 수렴 |
| **충**분성 | 모수에 대한 정보 최대 |

#### 그룹 2: 불편추정량 유형 `평분비`

| 요소 | 설명 |
|:-----|:-----|
| 표본**평**균 | E(X̄) = μ |
| 표본**분**산 | E(S²) = σ² (n-1로 나눔) |
| 표본**비**율 | E(P̂) = p |

---

### ⭐ 차별점 키워드

**"n-1로 나누는 이유 = 자유도"** - 표본분산이 모분산의 불편추정량이 되려면 n이 아닌 n-1로 나눠야 함

| 항목 | 내용 |
|:-----|:-----|
| **누가** | 통계학자, 데이터 분석가 |

| **왜** | 체계적인 과대/과소 평가 방지 |

### 📌 핵심 암기 (Quick Reference)

{: .highlight }
> **불편추정량**: 기대값, 모수일치, 편향제거 → 비편향 표본통계량
> - (점추정 준거) `불효일충` (불편성, 효율성, 일치성, 충분성)
> - (유형) `평분비` (표본평균, 표본분산, 표본비율)
> - (조건) `기편표` (기대값조건, 편향제거, 표본크기)
> - (표본수집) `무독표` (무작위표본, 독립성유지, 표본크기적절)

### 🏗️ 구성요소

#### 그룹 1: 점추정의 4가지 준거 `불효일충`

| 준거 | 설명 | 수식/조건 |
|:-----|:-----|:---------|
| **불**편성(=불편추정량) | 추정량의 기대값이 모수와 일치 | E(θ̂) = θ |
| **효**율성(=유효성) | 여러 불편추정량 중 분산이 가장 작음 | 분산이 작은 추정량이 더 정확 |
| **일**치성 | 표본크기 n→∞ 일 때 추정량이 모수에 수렴 | lim(n→∞) θ̂ = θ |
| **충**분성 | 추정량이 모수에 대한 최대 정보 제공 | 충분통계량 조건 |

---

#### 그룹 2: 불편추정량의 조건 `기편표`

| 구분 | 조건 | 설명 | 수식 |
|:-----|:-----|:-----|:-----|
| 조건 | **기**대값 조건 | 추정량의 기대값이 모수와 일치하는 조건 | E(θ̂) = θ |
| 조건 | **편**향 제거 | 추정량과 모수의 차이인 편향이 0이 되도록 제거 | Bias = E(θ̂) - θ = 0 |
| 조건 | **표**본크기 | 표본의 크기가 충분히 커야 모수와 표본 통계량 편차 최소화 | n ≥ 30 (대수의 법칙) |

---

#### 그룹 3: 불편추정량의 유형 `평분비`

| 유형 | 모수 | 불편추정량 | 공식 |
|:-----|:-----|:----------|:-----|
| 표본**평**균 | μ (모평균) | X̄ (표본평균) | X̄ = (1/n) × Σ(Xᵢ) |
| 표본**분**산 | σ² (모분산) | S² (표본분산) | S² = (1/(n-1)) × Σ(Xᵢ - X̄)² |
| 표본**비**율 | p (모비율) | P̂ (표본비율) | P̂ = x/n |

### 📊 표본분산 공식과 자유도

```
┌─────────────────────────────────────────────────────────────────────┐
│                        표본분산 공식                                  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│   표본평균의 불편추정량:                                               │
│                                                                      │
│                         X₁ + X₂ + ... + Xₙ                          │
│        E(X̄) =  μ,   X̄ = ─────────────────── = a                    │
│                                n                                     │
│                                                                      │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│   표본분산의 불편추정량 (n-1로 나눔):                                   │
│                                                                      │
│                       (X₁-a)² + (X₂-a)² + ... + (Xₙ-a)²             │
│        E(S²) = σ²,   S² = ─────────────────────────────────          │
│                                      n - 1                           │
│                                                                      │
│   ※ n으로 나누면 분산이 과소추정됨 → n-1로 나눠야 불편추정량            │
│                                                                      │
├─────────────────────────────────────────────────────────────────────┤
│   자유도 (Degrees of Freedom)                                        │
│                                                                      │
│   ┌───┐  ┌───┐  ┌───┐                                               │
│   │ 1 │  │ 2 │  │ 3 │  ← 3개의 변수, 평균이 6일 때                   │
│   └───┘  └───┘  └───┘                                               │
│      └──────┬──────┘                                                 │
│          결정됨                                                       │
│                                                                      │
│   • 2개의 변수가 결정되면 나머지 1개는 자동으로 결정                    │
│   • 자유롭게 선택할 수 있는 변수의 수 = n - 1                          │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### 📋 표본 수집 조건 `무독표`

| 구분 | 조건 | 설명 |
|:-----|:-----|:-----|
| 표본수집 | **무**작위 표본 필요 | 표본이 무작위로 선택되지 않으면 모집단을 대표하지 못함 |
| 표본수집 | **독**립성이 유지 | 표본 데이터 간 독립성이 없으면 특정 방향으로 편향될 가능성 |
| 표본수집 | **표**본 크기가 적절 | 표본이 너무 작으면 추정량이 불안정하여 모집단 정확히 반영 어려움 |

---

### 📋 모수 일치성 조건

| 조건 | 수식 | 설명 |
|:-----|:-----|:-----|
| 기대값이 모수와 동일 | E[θ̂] = θ | 불편추정량은 평균적으로 모집단의 모수를 정확하게 추정 |
| 편향(Bias)이 없음 | Bias = 0 | 편향이 존재하면 추정값이 체계적으로 과대 또는 과소 평가 |

---

### ⭐ 차별점 키워드 (가산점 포인트)

{: .important }
> **자유도(Degrees of Freedom)와 n-1**
> - 표본분산을 n으로 나누면 모분산을 **과소추정**
> - n-1로 나눠야 모분산의 **불편추정량**이 됨
> - 자유도: 평균이 고정되면 n개 중 n-1개만 자유롭게 선택 가능
> - 불편성, 일치성, 유효성, 충분성을 고려해 표본통계량 중에 모수를 추정하기 좋은 추정량을 선택
