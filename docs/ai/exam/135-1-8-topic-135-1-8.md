### 🔑 핵심 키워드 3개

| 키워드 | 설명 |
|:------|:-----|
| **기대값** | 추정량의 평균값 (여러 번 추정했을 때의 평균) |
| **모수** | 모집단의 실제 특성값 (평균, 분산 등) |
| **편향 없음** | 추정량의 기대값이 모수와 정확히 일치 |

### 📖 등장배경

| 구분 | 내용 |
|:-----|:-----|
| **과거** | 전교생 키를 다 재지 않고 일부만 재서 전체 평균을 추정하고 싶음 |
| **문제** | 표본으로 구한 평균이 전체 평균과 얼마나 비슷할지 알 수 없음 |

### 📝 개념 정의

| 개념 | 정의 | 쉬운 비유 |
|:-----|:-----|:---------|
| **불편추정량** | 기대값이 모수와 일치하는 추정량 | 과녁 중심을 평균적으로 정확히 맞추는 것 |

### 🧩 기술요소 (2그룹 × 4개)

#### 그룹 1: 점추정 4가지 준거 `불효일충`

| 요소 | 설명 |
|:-----|:-----|
| **불**편성 | 기대값이 모수와 일치 |
| **효**율성 | 분산이 가장 작음 |
| **일**치성 | n이 커지면 모수에 수렴 |
| **충**분성 | 모수에 대한 정보 최대 |

#### 그룹 2: 불편추정량 유형 `평분비`

| 요소 | 설명 |
|:-----|:-----|
| 표본**평**균 | E(X̄) = μ |
| 표본**분**산 | E(S²) = σ² (n-1로 나눔) |
| 표본**비**율 | E(P̂) = p |

### ⭐ 차별점 키워드

**"n-1로 나누는 이유 = 자유도"** - 표본분산이 모분산의 불편추정량이 되려면 n이 아닌 n-1로 나눠야 함

### 🧾 6하원칙 요약

| 항목 | 내용 |
|:-----|:-----|
| **누가** | 통계학자, 데이터 분석가 |
| **언제** | 모집단 특성을 표본으로 추정할 때 |
| **어디서** | 추정통계, 머신러닝, 품질관리 |
| **무엇을** | 편향 없이 모수를 추정하는 통계량 |
| **어떻게** | E(θ̂) = θ 조건 만족 |
| **왜** | 체계적인 과대/과소 평가 방지 |

### 📌 핵심 암기 (Quick Reference)

{: .highlight }
> **불편추정량**: 정확한 모수 추정 필요 → 기대값, 모수일치, 편향제거 → 비편향 표본통계량
> - (점추정 준거) `불효일충` - 불편성, 효율성, 일치성, 충분성
> - (유형) `평분비` - 표본평균, 표본분산, 표본비율
> - (조건) `기편표` - 기대값조건, 편향제거, 표본크기
> - ⭐ **차별점**: 자유도(Degrees of Freedom)와 n-1: > - 표본분산을 n으로 나누면 모분산을 **과소추정**

## 🧠 개념 영역


### 🔑 핵심 키워드 3개

| 키워드 | 설명 | 예시 |
|:------|:-----|:-----|
| **기대값 = 모수** | E(θ̂) = θ 조건 | 추정량의 평균이 모수와 일치 |
| **편향(Bias) = 0** | 추정량과 모수의 차이가 평균적으로 0 | Bias = E(θ̂) - θ = 0 |
| **자유도(n-1)** | 표본분산을 불편추정량으로 만들기 위한 조건 | S² = Σ(Xᵢ-X̄)² / (n-1) |

| **비즈니스 요구** | 표본 통계량이 모수를 체계적으로 과대/과소 평가하면 안 됨 |

## 🏗️ 기술 영역

<div class="exam-tech-block" markdown="1">

### 🏗️ 구성요소 (핵심 암기법 상세) `불효일충` / `평분비`

#### 그룹 1: 점추정의 4가지 준거 `불효일충`
{: .highlight-purple }

| 암기 | 준거 | 설명 | 수식/조건 |
|:-----|:-----|:-----|:---------|
| **불** | 불편성 | 추정량의 기대값이 모수와 일치 | E(θ̂) = θ |
| **효** | 효율성(유효성) | 여러 불편추정량 중 분산이 가장 작음 | 분산이 작은 추정량이 더 정확 |
| **일** | 일치성 | 표본크기 n→∞ 일 때 추정량이 모수에 수렴 | lim(n→∞) θ̂ = θ |
| **충** | 충분성 | 추정량이 모수에 대한 최대 정보 제공 | 충분통계량 조건 |

#### 그룹 2: 불편추정량의 유형 `평분비`
{: .highlight-purple }

| 암기 | 유형 | 모수 | 불편추정량 | 공식 |
|:-----|:-----|:-----|:----------|:-----|
| **평** | 표본평균 | μ (모평균) | X̄ | X̄ = (1/n) × Σ(Xᵢ) |
| **분** | 표본분산 | σ² (모분산) | S² | S² = (1/(n-1)) × Σ(Xᵢ-X̄)² |
| **비** | 표본비율 | p (모비율) | P̂ | P̂ = x/n |

## ⭐ 차별점 키워드 (가산점 포인트)

{: .important }
> **자유도(Degrees of Freedom)와 n-1**
> - 표본분산을 n으로 나누면 모분산을 **과소추정**
> - n-1로 나눠야 모분산의 **불편추정량**이 됨
> - 자유도: 평균이 고정되면 n개 중 n-1개만 자유롭게 선택 가능

#### 📊 표본분산 공식과 자유도

```
┌─────────────────────────────────────────────────────────────────────┐
│                        표본분산 공식                                  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│   표본평균의 불편추정량:                                               │
│                                                                      │
│                         X₁ + X₂ + ... + Xₙ                          │
│        E(X̄) =  μ,   X̄ = ─────────────────── = a                    │
│                                n                                     │
│                                                                      │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│   표본분산의 불편추정량 (n-1로 나눔):                                   │
│                                                                      │
│                       (X₁-a)² + (X₂-a)² + ... + (Xₙ-a)²             │
│        E(S²) = σ²,   S² = ─────────────────────────────────          │
│                                      n - 1                           │
│                                                                      │
│   ※ n으로 나누면 분산이 과소추정됨 → n-1로 나눠야 불편추정량            │
│                                                                      │
├─────────────────────────────────────────────────────────────────────┤
│   자유도 (Degrees of Freedom)                                        │
│                                                                      │
│   • 평균이 고정되면 n개 중 n-1개만 자유롭게 선택 가능                   │
│   • 자유도 = n - 1                                                    │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

#### 📋 표본 수집 조건 `무독표`

| 구분 | 조건 | 설명 |
|:-----|:-----|:-----|
| 표본수집 | **무**작위 표본 필요 | 표본이 무작위로 선택되지 않으면 모집단을 대표하지 못함 |
| 표본수집 | **독**립성이 유지 | 표본 데이터 간 독립성이 없으면 특정 방향으로 편향될 가능성 |
| 표본수집 | **표**본 크기가 적절 | 표본이 너무 작으면 추정량이 불안정 |

### 📊 불편추정량 vs 편향추정량 비교

| 구분 | 불편추정량 | 편향추정량 |
|:-----|:----------|:----------|
| **기대값** | E(θ̂) = θ (모수와 일치) | E(θ̂) ≠ θ (모수와 불일치) |

| **특징** | 평균적으로 정확한 추정 | 과대/과소 추정 가능 |
| **예시** | 표본평균, 표본분산(n-1) | n으로 나눈 표본분산 |
