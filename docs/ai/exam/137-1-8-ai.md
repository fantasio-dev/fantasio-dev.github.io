### ⭐ 차별점 키워드

**"신뢰성(Trustworthiness) 중심"** - 성능만 아니라 책임·안전·공정·설명까지 종합 평가

### 🧾 6하원칙 요약

| 항목 | 내용 |
|:-----|:-----|
| **누가** | 조직/기업이(또는 인증기관이) |
| **언제** | 개발/도입/운영 전후로 |
| **어디서** | AI 모델·시스템·거버넌스 전 영역에서 |
| **무엇을** | 신뢰성 요구사항을 |
| **왜** | AI 리스크를 줄이고 책임 있는 활용을 위해 |
| **어떻게** | `위거신추` 기반으로 평가하고 인증 |

#### 📋 인증 기준 (요구사항 1~15)

| 4 | AI 시스템 추적가능성 및 변경이력 확보 | 데이터 출처/변환 정보 기록 |

#### 📋 CAT 1.0 vs CAT 2.0

| 구분 | CAT 1.0(초기) | CAT 2.0(고도화) |
|:-----|:-------------|:---------------|
| 평가 방식 | 문서·절차 중심 심사 | 실제 운영환경 기능/성능 시험 강화 |
| 적용 시점 | 초기 도입 단계 | 2025년 4월부터 본격 적용 |
| 기준 기반 | 표준/가이드라인 참고 | 국제표준 기반 평가 강화 |
