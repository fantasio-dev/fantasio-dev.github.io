### 📝 개념 정의

> **유전 알고리즘(Genetic Algorithm)**: 자연세계의 진화현상인 유전학의 원리에 근거하여, 세대를 거치면서 적자생존을 통해 점진적으로 최적해를 탐색해가는 최적화 문제 해결 알고리즘

| 개념 | 쉬운 비유 |
|:-----|:---------|
| **유전 알고리즘** | 더 좋은 자식을 낳기 위해 우수한 부모끼리 결혼시키는 것 |
| **적합도** | 시험 점수 (높을수록 좋음) |

## 🎯 기술사 수준 설명
{: #-기술사-수준-설명}

### 📌 핵심 암기 (Quick Reference)

{: .highlight }
> **유전 알고리즘**: 진화원리, 적자생존, 전역최적화 → 메타휴리스틱 최적화 알고리즘
> - (구성요소-기본) `염유개인` - 염색체, 유전자, 개체군, 인코딩
> - (구성요소-제어) `진매종` - 진화연산자, 매개변수, 종료조건
> - (절차) `초적선교변대반` - 초기화, 적합도평가, 선택, 교차, 변이, 대체, 반복
> - (선택기법) `룰순토` - 룰렛휠, 순위, 토너먼트
> - (교차기법) `단다균산` - 단일점, 다점, 균등, 산술
> - ⭐ **차별점**: 유전 알고리즘의 핵심 특징: > - **전역 최적화(Global Optimization)에 적합**: 지역 최적해에 빠지지 않음

### 📋 유전 알고리즘의 절차 `초적선교변대반`

| 순서 | 절차 | 설명 |
|:-----|:-----|:-----|
| 1 | **초**기화(Initialization) | 문제의 해를 표현하는 염색체들의 초기 개체 집단을 임의로 생성, 랜덤한 유전자를 적당한 개수만큼 준비 |
| 2 | **적**합도(Fitness) 평가 | 각 염색체의 품질을 적합도 함수를 이용하여 점수 계산, 문제의 목적함수에 따라 평가 |
| 3 | **선**택(Selection) | 현재 세대의 개체들 중 적합도가 높은 개체를 부모로 선택, 룰렛휠 선택, 순위 선택, 토너먼트 선택 등 적용 |
| 4 | **교**차(Crossover) | 선택된 부모들 간에 유전 정보를 교환하여 자식해 생성, 단일점 교차, 다중점 교차, 균등 교차, 산술교차 등 적용 |
| 5 | **변**이(Mutation) | 일부 자식해의 유전자 값을 랜덤하게 변형하여 다양성 유지, 비트 반전 돌연변이, 교환 돌연변이, 삽입 돌연변이 등 적용 |
| 6 | **대**체(Replace) | 새롭게 생성된 해들을 기존 세대와 교체하여 다음 세대 구성 |
| 7 | **반**복(Loop) | 종료 조건 충족 시까지 알고리즘 반복 수행, 거의 모든 유전자가 같아졌고 변화가 거의 없을 때까지 반복 |

## 나. 유전 알고리즘의 최적화 방법

### 📋 가. 성능 향상을 위한 최적화 방법

#### 1. 적합도 함수 설계(Fitness Function Design)

| 방법 | 설명 | 특징 |
|:-----|:-----|:-----|
| 적합도 함수 | 문제 목표를 수치화하여 적합도 평가 | 문제 정의에 따라 GA 성능 크게 좌우 |
| 정규화/스케일링 | 적합도 값의 범위 조정 | 선택 압력 조절 |
| 벌점 함수 | 제약 조건 위반 시 페널티 부여 | 제약 최적화 문제에 활용 |

---

#### 2. 선택 전략(Selection Strategy) `룰순토`

| 기법 | 설명 | 특징 |
|:-----|:-----|:-----|
| **룰**렛 휠 선택 | 적합도에 비례하여 선택 확률을 차등 적용하여 선택 | 적합도 비율만큼 확률적 선택 |
| **순**위 선택 | 적합도 순으로 순위를 매긴 후 가장 높은 해부터 순차적으로 선택 | 선택 압력 일정, No selection pressure |
| **토**너먼트 선택 | 개체군 중 일정 개수 개체 임의 선택, 가장 높은 적합도 개체 선택 | 임의의 N개 중 최적 개체 선택 |

---

#### 3. 교차 전략(Crossover Strategy) `단다균산`

| 기법 | 설명 | 특징 |
|:-----|:-----|:-----|
| **단**일점 교차 | 부모 염색체를 한 지점에서 나누고 교환하여 자손 생성 | 특정 지점을 기준으로 유전자 교차 |
| **다**중점 교차 | 두 개 이상의 지점에서 나누고 교환하여 자손 생성 | 더 다양한 조합 가능 |
| **균**등 교차 | 각 유전자를 독립적으로 부모 중 하나에서 랜덤 선택해 자손 생성 | 각 유전자에 대해 난수로 결정 |
| **산**술 교차 | 수치형 유전자의 가중 평균으로 자손 생성 | 현재 유전자에 특정 산술 연산 적용 |

---

#### 4. 돌연변이 연산(Mutation Operator)

| 기법 | 설명 | 특징 |
|:-----|:-----|:-----|
| 비트 반전 돌연변이 | 유전자 값을 반대로 변경 (0↔1) | 이진 인코딩에 적합 |
| 교환 돌연변이 | 두 유전자 위치를 서로 교환 | 순열 표현에 적합 |
| 삽입 돌연변이 | 유전자를 다른 위치에 삽입 | 순서 기반 문제에 활용 |
| 비균등 변이 | 세대가 진행될수록 변이 범위 축소 | 미세 조정에 활용 |

---

### 📋 나. 다양성 확보를 위한 최적화 방법

| 구분 | 방법 | 설명 |
|:-----|:-----|:-----|
| 다양성 확보 | **엘리트 전략(Elitism)** | 현재 세대의 우수한 개체를 다음 세대로 그대로 전달, 최적 해 손실 방지, 수렴 속도 개선 |
| 다양성 확보 | **대체 전략(Replacement)** | 새로 생성된 자식이 부모 세대를 어떻게 대체할지 정의하는 방법 |
| 다양성 확보 | **크라우딩 전략** | 유사도 기준으로 대체하여 다양성을 확보하는 방법 |

---

### 📋 다. 자원 효율화 전략

| 구분 | 방법 | 설명 |
|:-----|:-----|:-----|
| 자원 효율화 | **하이브리드 전략** | 유전 알고리즘과 다른 최적화 기법을 결합 (예: GA + 지역 탐색 = Memetic Algorithm) |
| 자원 효율화 | **종료 조건 설정** | 알고리즘 종료 시점을 합리적으로 설정 (예: 적합도 변화 없음, 최대 세대 수 도달) |
| 자원 효율화 | **파라미터 조정(Parameter Tuning)** | 집단 크기, 교차/돌연변이 확률 조정, 수렴 속도와 탐색 품질에 영향 |

### 📊 유전 알고리즘 적용 사례

| 분야 | 사례 | 설명 |
|:-----|:-----|:-----|
| **최적화** | 최단 경로 찾기, 배차 문제, 재고 관리 | 최적의 해를 찾는 데 사용 |
| **머신러닝** | 분류, 회귀, 추천 시스템, 하이퍼파라미터 최적화, 피처 선택 | 기계 학습 모델을 학습시키는 데 사용 |
| **과학연구** | 약물 개발, 재료 과학, 유전학 | 과학 연구에서 새로운 발견을 하는 데 사용 |
| **공학설계** | 구조 설계, 회로 최적화 | 설계 최적화에 활용 |
| **스케줄링** | 작업 스케줄링, 생산 계획 | 일정 최적화에 활용 |
| **경로계획** | 로봇 경로 최적화, 물류 배송 최적화 | 경로 탐색에 활용 |

### ⭐ 차별점 키워드 (가산점 포인트)

{: .important }
> **유전 알고리즘의 핵심 특징**
> - **전역 최적화(Global Optimization)에 적합**: 지역 최적해에 빠지지 않음
> - **메타휴리스틱 기법**: 복잡한 산업 기술분야에서 복잡한 문제를 해결하기 위해 활용
> - 탐색 성능 향상, 조기 수렴 방지, 해의 다양성 유지, 계산 자원 효율화 등을 위해 최적화 방법 적용 중요
> - 최적화 목적, 적합도 분포, 계산 비용 등을 고려하여 적절한 최적화 방법을 선택하는 것이 중요

### 📈 상위 토픽 계층도

```
최적화 알고리즘 (Optimization Algorithms)
├── 전통적 최적화
│   ├── 경사 하강법 (Gradient Descent)
│   ├── 뉴턴 방법
│   └── 라그랑주 승수법
├── 메타휴리스틱 최적화
│   ├── 진화 알고리즘
│   │   └── 유전 알고리즘 (Genetic Algorithm) ⭐
│   │       ├── 구성요소: 염색체, 유전자, 개체군, 인코딩
│   │       ├── 절차: 초기화→적합도평가→선택→교차→변이→대체→반복
│   │       └── 최적화: 선택전략, 교차전략, 엘리트전략
│   ├── 담금질 기법 (Simulated Annealing)
│   ├── 타부 서치 (Tabu Search)
│   └── 입자 군집 최적화 (PSO)
└── 하이브리드 최적화
    └── Memetic Algorithm (GA + 지역 탐색)
```
