# 퓨샷러닝(Few-Shot Learning)
{: .fs-8 }

학습 기법
{: .label .label-green }

---

## 🎯 기술사 수준 설명

### 📌 핵심 암기 (Quick Reference)

{: .highlight }
> **퓨샷러닝**: 소량의 데이터로 효과적인 학습을 하는 학습 방식
> - (유형) `제원퓨` 제로샷, 원샷, 퓨샷
> - ⭐ **차별점**: 제로샷은 관측되지 않은 클래스 학습, 원샷은 1개 샘플, 퓨샷은 소수 샘플

---

## 핵심 키워드

`제로샷` `원샷` `퓨샷` `적은 샘플` `빠른 학습`

---

## 정의/개념

소량의 데이터로 효과적인 학습을 하는 학습 방식

---

## 유형 `제원퓨`

| 암기 | 유형 | 설명 | 예시 |
|:-----|:-----|:-----|:-----|
| **제** | 제로샷러닝 | 한번도 관측되지 않은 클래스 학습 | 호랑이→사자→라이거 |
| **원** | 원샷러닝 | 하나의 샘플만 주어짐 | 한 개의 호랑이, 두 번째는 맞춤 |
| **퓨** | 퓨샷러닝 | 클래스별 소수 샘플 | 적은 수의 호랑이, 다음부터 맞춤 |

---

## 비교

| 유형 | 샘플 수 | 특징 |
|:-----|:--------|:-----|
| **제로샷** | 0개 | 관측되지 않은 클래스 분류 |
| **원샷** | 1개 | 하나의 예시로 학습 |
| **퓨샷** | 수 개 | 소수의 예시로 학습 |
